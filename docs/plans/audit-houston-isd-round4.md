## 剩余优化项

### P1 — Meta Description 压缩

**当前（过长，约 180+ 字符）：**
```
Houston ISD Calendar 2026-2027: first day Aug 10... Download PDF or sync to Google Calendar.
```

**建议（约 150 字符）：**
```
Houston ISD Calendar 2026-2027 with holidays, spring break and key dates.
Download the official PDF or add the calendar to Google Calendar.
```

### P1 — Quick Facts "Same as Fort Worth" 替换

价值低，建议换为以下任一：
- `Teacher Last Day: Jun 1`
- `Spring Break: Mar 8–12`
- `Semester 2 Begins: Jan 5`

### P1 — Hurricane FAQ 措辞更谨慎

**当前（过于绝对）：**
> Texas law requires school districts to make up instructional days...

**建议：**
> HISD may use built-in instructional minutes, makeup days, or state-approved waivers after weather closures. Families should check official district updates if severe weather disrupts the school year.

### P2 — 192 Teacher Work Days 补充来源

在 FAQ 回答中增加：
> Based on the official HISD teacher work calendar.

HTML 仍为 Nuxt/Vite 开发环境输出（`@vite/client`、本地路径等），生产上线前须用 `nuxt build` 构建。
