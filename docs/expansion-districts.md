# Expansion Districts — 200 to Add

Districts ranked by student enrollment within each state. The 25 already-live districts are excluded.

> **Note:** Calendar URLs are estimated based on official district websites — verify the exact PDF download path before downloading.

---

## How to add a new district

1. Download the official calendar PDF from the URL below
2. Create `content/districts/<slug>.json` with district metadata
3. Send both PDF file paths in chat, e.g.:
   > `/path/to/2025-2026.pdf /path/to/2026-2027.pdf 这两个是 XXXX 的`
4. Claude will parse and create the calendar JSON files
